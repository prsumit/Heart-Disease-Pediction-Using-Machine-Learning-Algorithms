Heart Disease Prediction Web Application (Machine Learning-Based)
This project is a **machine learning–based web application** developed to predict the risk of heart disease in patients using their medical data. Heart disease is a serious and life-threatening condition, and early prediction can help in timely diagnosis and treatment.

Project Overview
The system allows users to enter their health-related information such as age, blood pressure, cholesterol level, heart rate, and other medical parameters. Based on this input, the application uses trained machine learning models to predict whether a person is at risk of heart disease.
Both **patients and doctors** can access the results through the web platform. Doctors can analyze patient data more efficiently, while patients can track their health reports digitally.
Additionally, the application includes a **location-based feature**, where users can find hospitals in their city and book appointments with doctors directly through the platform. This improves accessibility to healthcare services.

Machine Learning Algorithms Used
Multiple algorithms were implemented and compared to identify the most accurate model:
* Logistic Regression – Accuracy: 85%
* Decision Tree – Accuracy: 82%
* Random Forest – Accuracy: 90%
* Support Vector Machine (SVM) – Accuracy: 88%
* K-Nearest Neighbors (KNN) – Accuracy: 84%
* Naïve Bayes – Accuracy: 80%

Among these, Random Forest provided the highest accuracy and best performance for prediction in this project.
Key Features
* Predicts heart disease risk using ML models
* Compares multiple algorithms to find the best accuracy
* User-friendly web interface
* Separate access for patients and doctors
* Secure storage and viewing of medical data
* City-based hospital search and doctor appointment booking

*Conclusion*
This project demonstrates how machine learning can be effectively used in the healthcare sector for early detection of heart disease. By integrating prediction with a web-based system and hospital booking feature, it provides a complete digital healthcare solution that can save time, improve diagnosis, and potentially save lives.
